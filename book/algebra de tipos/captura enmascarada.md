# Captura con enmascaramiento

El tipo `Mut Bijection Matrix[4, 4] Complex` puede ser capturado por:

* `Complex`
* `Matrix Complex`
* `Var .. Complex`
* `Bijection .. Complex`
* `Matrix[4, $N] Number`

```
DEF 

FUNCTION
::name fft_inplace
::doc
	Funcion foo
::param
	value: Bijection[Nat] N
::where
	W: Nat
	H: Nat
	N: Number
::body

```
